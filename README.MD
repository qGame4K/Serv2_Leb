# Контрольная работа №2


## Описание проекта

В данном репозитории представлены решения задач контрольной работы №2 по созданию серверных приложений на базе фреймворка FastAPI. Проект разделен на несколько логических модулей согласно выданным заданиям.

## Структура репозитория

* `task_3_1.py` — **Задание 3.1**: Конечная точка `POST /create_user` для регистрации пользователей с валидацией данных через Pydantic (проверка email и возраста).
* `task_3_2.py` — **Задание 3.2**: Маршруты для работы с товарами (`GET /product/{product_id}` и `GET /products/search`). Обработка конфликтов маршрутов.
* `task_5.py` — **Задания 5.1 - 5.3**: Реализация безопасной аутентификации на основе файлов cookie. Включает генерацию HMAC-подписей и механизм динамического продления времени жизни сессии (максимум 5 минут, обновление после 3 минут).
* `task_5_4.py` — **Задание 5.4**: Извлечение HTTP-заголовков `User-Agent` и `Accept-Language` напрямую из запроса.
* `task_5_5.py` — **Задание 5.5**: Оптимизация работы с заголовками (принцип DRY) с использованием модели Pydantic `CommonHeaders` и FastAPI `Depends`. Добавление кастомного заголовка ответа `X-Server-Time`.
